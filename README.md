# DoseLab

DoseLab is an open-source, local-first, cross-platform medication self-observation tool.

It is intended for users who have ongoing medication needs and want to understand their dosing rhythm through pharmacokinetic modeling, dose logs, and concentration curve visualization.

## Status

Early concept stage.

## Product Direction

- Local-first medication tracking
- Pharmacokinetic concentration curve visualization
- Therapeutic window overlays
- Dose logs and adherence insights
- Transparent model parameters
- FDA data source integration for medication reference data
- Cross-platform Material 3 interface
- User-owned data with JSON import/export

See `docs/FDA_API.md` for the initial openFDA integration notes.

## Principles

- Local-first by default
- No medical advice
- Model-transparent
- Source-transparent
- User-owned data
- Cross-platform
- Open-source

## Disclaimer

DoseLab is intended for personal recording, simulation, and visualization only. It is not a medical device and does not provide diagnosis, treatment, prescription, or dosage adjustment advice.

Model results are estimates based on user input and public parameters. They may differ significantly from real blood concentrations. Do not adjust, stop, or start medication based on this software. Always consult a qualified clinician or pharmacist for medical decisions.
